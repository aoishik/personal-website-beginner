AOILED

AOILED is a smart LED lighting system that allows users to control the brightness and color of their lights remotely. It also has a microphone for audio reactive lighting. It is mainly for 12v led strips and can be controlled via a web interface or the WLED mobile app. It is built using ESP32 and WLED firmware.
